Why Anki?

Because Anki is simple.

Okay I lied. Anki supports TeX. That is so freaking cool. And it support MathJax and it support html.

I recommand have some prequistion of html before editing the content of Anki card.

LaTeX in Anki

It is a pitty that Anki does not support TeX, but maybe there is some plugin to that. Anyway, the compiler is LaTeX and Anki use dvisvgm to convert the .dvi file to .svg file and the letter is displayed when you browsing the card. Neat.

Install

You need to install the TeX live. You may only install latex and plain tex which only require space about 200 MB. You may install dvisvgm manually using the TeX live manager.

Use

The relative code is in the templates of the cards. You may change it to import package or define some macros.

You can use standalone document class to produce a neat piece of latex output.

A universal solution to editing

By default, one could only use the interior text-piece-of-shit-editor to edit the content of the card or the html code of it.

However, you may use text editor anywhere to solve this problem once for all.
