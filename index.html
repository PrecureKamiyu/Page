About learning Japanese

Are you interested about the Japanese? If you are, there is something interesting about learning Japanese, that is 学校语法和教育语法.

教育语法是面对非母语学习者的, 在标日中, 在单词表中, 动词为ます形, 但其在学校语法中称为连体形 + ます助动词, 类比于英语里面, can should 等助动词后面需加上原型, 而 need 可以加不定式. 同理, ない形也是如此. 更比如, 我们能见到复合动词, 其为两个动词的链接, 形式为连体形 + 动词

我们能看见这里的分歧, 现在的问题是如何选择. 得到的建议是, 先学教育语法, 学到中等水平之后再深入接触学校语法. 各个人的意见各不相同, 并且有人指出, 教学领域里其实也是混乱不堪, 没人搞得懂哪个更为合适.

About Japanese in TeX

pTeX is invented for typesetting Japanese. And UpTeX is the extended version of pTeX. Both of them may have been viewed as Legacy for typesetting Japanese for LuaTeX is doing a pretty good job.

Before you try to formatting some Japanese yourself, you may want to check out for this article about jlreq. Basically, it tells about the standard when formatting Japanese. You may skip it since it is too long. Anyway, you need to know that there is such thing as jlreq.

You may begin with a simple .tex file

       \documentclass{jlreq}
       \begin{document}
       こんにちは。
       \end{document}

And compile the .tex file with uplatex. And you will get a .dvi file, whose name stands for device indenpendent, which is one of the feature of .pdf file, since it is designed that the same thing is displayed on different devices and you may guess that .dvi plays the similar role before .pdf is born. Stupid Japanese use it because they can, for their devices haven't been renewed for decades. Your device already have been installed with .dvi reader if you have installed TeX Live. A tool dvipdfmx is also installed which is used to, as its name has shown, to convert .dvi file into a .pdf file.

However, things get interesting when you try to open the output file, viz., the .dvi file. You just can't open it, or you may be able to open it since there could be something wrong with my device. Anyway, what is happening is that .dvi file does not have the information of font. When uplatex compile the .tex file, it use .dvi file to store information including the position of the characters, but not the specific font that would be displayed. Later, dvipdfmx map the font. And further more, you can actually change the font not by changing the .tex file, but by applying an additional map when using dvipdfmx.

Consequently, you can formatting Hanzi using upLaTeX. And there is vertical typesetting mode in upLaTeX. For jlreq class, just add parameter tate to the document class, viz., \documentclass[tate]{jlreq}. Thus you can formatting Hanzi vertically (after reading this article).
