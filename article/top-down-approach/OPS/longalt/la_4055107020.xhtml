In this illustration, we see the same BSS from the previous figure, connected this time to a series of switches. Above the BSS is a dotted line; area above the line is indicated as "Control plane" while area below the line is "Data plane."

Within the BSS is a small device, labeled "UE" and operating in the control plane. Each base transceiver station is connected to a single network controller, labeled "eNodeB" and operating in the control plane. The eNodeB is connected to a network of routers operating in the data plane and arranged in a square. The top left router is connected to a network controller labeled "MME," operating in the control plane. The top right router is connected to a network controller labeled "HHS," operating in the control plane. The bottom right router is connected to a network controller labeled "S-GW" and another labeled "P-GW," both operating in the control plane. All of the devices operating in the control plane are connected one to the next by a dotted line.
