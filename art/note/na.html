Polynomial Interpolation

We are going to introduce two kinds of interpolation. One is called Lagrange interpolation. The other is called Newton's interpolation

Lagrange interpolation

The main idea is to use the basis functions that is \(l _i (x _{j}) = \delta _{i} ^j\). The linear composition of basis functions can construct a polynomial that agree with the original function on the given \(n + 1\) points.

\( \delta _{i} ^j\) is called Kronecker symbol. It equals to \(1\) if and only if \(i = j\).

Any way it is easy to express the \(l_{i}\) which is a polynomial and have \(n\) roots and they are \(x _{j}\), \(j \ne i\), which leads to that the \(l_{i}\) can expressed as the following: \[ \prod _{0 \le j \le n, j \ne i} \frac{x - x _{j}}{x _{i} - x _{j}} \]

There is a very good property \[ \sum _{ 0 \le i \le n } l _{i} (x) = 1 \]and there is more \[ \sum _ { 0 \le i \le n } x_ i ^ k l _{i} (x) = x ^k \]

The procedure of using Lagrange interpolation

Give a point, first use the expression of the \(l_{i}\) to calculate the value of the \(l_{i}\) on that point. And then we need to calculate the error function. Use the formula

\[ E(x) = \frac{f ^ {(n+1)}(\xi)}{ ( n +1) !} p_{n+1} (x) \]

where the \(p\) is something like \( (x - x _{0}) (x - x _{1}) \dots (x - x_{n})\). To tell the range of the \(E (x)\), we need to tell the maxium of the functions relating.
