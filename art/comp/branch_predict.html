Exceptions

Exceptions occur when, for example, you get an unknown instruction. CPU needs to deal with it. How? Use the SEPC (Supervisor Exceptions Program Counter) and SCAUSE (supervise cause).

SEPC store the value of PC when exceptions happen. SCAUSE store the type of an exception when it happens.

Moreover, we want to rewind the state. So it is important that we don't reach WB stage. To do that we use an additional signal called EX.Flush. That is to say, if an instruction reach WB stage, the result of the instruction is written to memory or register, then we can't easily rewind, for we shall use another register to copy the value of the old one.
