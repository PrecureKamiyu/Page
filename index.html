\(\bf\TeX\)

What you should do if you want to learn TeX:

  1. Download TeX Live on your linux machine
  2. Run texdoc lshort-zh-cn in your terminal
  3. Read it
  4. Write your homework with LaTeX
  5. Read the TeXbook and learn some cool things about TeX

\(\bf\TeX\text{book}\)

\(\TeX\)book is not just about \(\TeX\). It also tells you about what is typesetting. I didn't know a thing about the spacing between some certain letters like V and A (the spacing between V and A should be adjusted when VA is next to each other.) before I knew about what kerning is in \(\TeX\).

\(\TeX\) helps you build a taste that tells you which is good typesetting and which is not. Clearly enough, html is not good enough for formatting text.

Maybe it is enough for one who is interested in \(\TeX\), but there is also some great source like TeX by Topic.

\(\bf \TeX\) in Vim

There are two plugin needed if you want to write \(\LaTeX\) in Vim. They are

  • Ultisnips
  • vimtex

Ultisnips makes you enter code more efficiently. Vimtex makes vim more smart in some details about \(\LaTeX\), for example, it can tell whether you are in math mode or not and most importantly, it can compile the tex file whenever you save the file. You don't have to compile the file manually after you make some modifications.
