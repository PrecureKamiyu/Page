Great books

what a load of shit!

  • Computer Organization and Design The Hardware Software Interface [RISC-V Edition] by David A. Patterson, John L. Hennessy: The writing style is quite different from other books. The author just keeps telling you things, but in a "burst mode". Instead of being blocks of contents, in this book, there are streams of contents.
  • Algebra: Chapter 0 by Paolo Allufi: The book can be used as a textbook of a course for graduate or under-graduate who want to improve themselves. Catergory is introduced in the first chapter. It is good, but requiring certain degree of mathematical maturity.
  • Computer Organization and Architecture: Designing for Performance by William Stallings: It is a book about the computer organization. It is worth noting that Tang's book has 借鉴了一下 this book. The basic structure is similar. And it it is clear that which book is more of value (as you can see, in Tang's book, there is a section about cache while there is a whole chapter to introduce cache in Stallings' book.)
  • Automata theory, Languages, and Computation: Its language is simple. The author is like your friend. Things may not be enough. But they are comfortable to read. And here is the link to the answer to the starred exercises in the book. PS. You may buy a 影印版.
  • 线性代数 by 李炯生: The author tells you all the details patiently. And the exercises are very interesting (and extremely difficult).
  • 近世代数引论 by 冯克勤: A book with less examples. But it tells things rigorously (not that rigorously, but very good enough). Similarly, the exercises are very interesting (and extremely hard).

Install fonts for texlive

You can just install the font system-wise. Or you can dump the font in the directory named texmf or something like that.

if the font is installed for the system, texlive can surely find the font. But you can use latter to install fonts just for texlive.

If you use the latter, use kpsewhich to check the directory you will need and use something starts with mk to refresh. Any way just search "install font for texlive" and you will find answer.
