MathJax

What happen when you insert math formula in markdown.

There is no such thing as "LaTeX in markdown". Most of the markdown previewers use MathJax or katex. I don't like MathJax. The default font it use is not good.

MathJax is not TeX.

Employ MathJax in your website

Just read the document of MathJax. Note that MathJax version 2 support different fonts while version 3 does not. If you are willing to change the font, you should check the documentation of MathJax version 2.

Change the font in your website

Note that there is actually three ways to render your formula:

  1. CHTML
  2. HTML+CSS
  3. SVG

where CHTML only supports one font. You may change the configuration to HTML+CSS or SVG if you want to change the font. The configuration is determined based on your source. Anyway, you may read the source code on current website for a check. After you change the configuration, read the documentation of MathJax font to change the configuration of SVG or HTML+CSS to display your preferred font.

Two Steps: 1. Change the rendering methods to SVG or the other; 2. Configure the font that SVG use.

There are four kinds of fonts available.

  1. TeX, which is ugly under certain circumstances.
  2. STIX, which is ugly under most of the circumstances.
  3. Asana-Math, which is ugly under most of the circumstances, just like STIX.
  4. Neo-Euler, which is pretty cool because you rarely see it, and Knuth use this font in his book Concrete Mathematics
  5. Latin-Modern, which is preferred since it suits CMU serif font best, but looks not so good because of SVG rendering.

The spacing between lines

It is true that the spacing between will be effected if you employ a math formula with an unusual height. For example, \(g ^2\) is a very simple formula. However the spacing is slightly bigger. I shall insert some test text to help you notice. It is true that what we often consider is true is not true under certain circumstances where the consideration is not truly reliable for something that is out of the consideration which cause that the thing is considered to true is not true.

Note: The spacing may be fine on some devices.

To advoid the bad spacing we shall adjust the size of the superscript in this scenario. You can use \scriptscriptstyle to produce a smaller superscript. For example, \(g^{\scriptscriptstyle 2}\). Maybe one should avoid writing long article when using MathJax.
