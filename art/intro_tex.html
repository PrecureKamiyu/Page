\(\bf\TeX\)

What you should do if you want to learn TeX:

  1. Download TeX Live on your linux machine
  2. Run texdoc lshort-zh-cn in your terminal
  3. Read it
  4. Write your homework with LaTeX
  5. Read the TeXbook and learn some cool things about TeX

\(\bf\TeX\text{book}\)

\(\TeX\)book is not just about \(\TeX\). It also tells you about what is typesetting.

\(\TeX\) helps you build a taste that tells you which is good typesetting and which is not.

\(\bf \TeX\) in Vim

There are two plugin needed if you want to write \(\LaTeX\) in Vim. They are

  • Ultisnips
  • vimtex

Ultisnips is a snippet plugin. You should write you own snippets. Videos on Youtube are great resource.

You can use Vimtex to auto-compile the current once you saved it. Of course, there are more functions.
