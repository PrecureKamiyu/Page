Install fonts for texlive

You can just install the font system-wise. Or you can dump the font in the directory named texmf or something like that.

if the font is installed for the system, texlive can surely find the font. But you can use latter to install fonts just for texlive.

If you use the latter, use kpsewhich to check the directory you will need and use something starts with mk to refresh. Any way just search "install font for texlive" and you will find answer.
