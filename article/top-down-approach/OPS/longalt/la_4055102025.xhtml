This figure uses a desktop computer and a series of servers to illustrate the steps that take place during a CDN interaction. The steps are numbered and correspond with the preceding text.

Step one, in which the user initially visits the site of NetCinema, is depicted as a blue arrow that extends diagonally upwards and to the right, pointing back and forth between the user's computer and a server labeled "www.NetCinema.com."

Step two is depicted as a blue arrow that points vertically to the right, from the user's computer to a server labeled "Local DNS server."

Step three is depicted as a blue arrow that extends further to the right and points back and forth between the Local DNS server and another server labeled "NetCinema authoritative DNS server."

Step four is depicted as a blue arrow that extends diagonally downward and to the right that points back and forth between the Local DNS server and another server labeled "KingCDN authoritative server."

Step five is depicted as a blue arrow that extends vertically leftward from the Local DNS server to the user's computer.

Step six is depicted as a blue arrow that extends from a fifth server, labeled "KingCDN content distribution server," to the user's computer.
