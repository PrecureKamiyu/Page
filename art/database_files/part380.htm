Further Reading

[Hennessy et al. (2017)] provides an excellent introduction to the area of computer ar- chitecture, including the topics of shared-memory architectures and cache coherency, parallel computing architectures, and cloud computing, which we covered in this chap- ter. [Gray and Reuter (1993)] provides the classic textbook description of transac- tion processing, including the architecture of client– server and distributed systems. [Ozsu and Valduriez (2010)] provides textbook coverage of distributed database sys- tems. [Abts and Felderman (2012)] provides an overview of data center networking.
